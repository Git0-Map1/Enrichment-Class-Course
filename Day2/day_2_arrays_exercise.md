# Array Exercises

## Exercise 1: Create an Array

Create an array called `colors` containing:

* `"red"`
* `"blue"`
* `"green"`
* `"yellow"`

Print the entire array.

---

## Exercise 2: Access Elements

Using the `colors` array:

1. Print the first element.
2. Print the last element.

---

## Exercise 3: Update an Element

Replace `"green"` with `"purple"`.

Print the updated array.

---

## Exercise 4: Find the Length

Create an array called `countries`:

```js
["Nepal", "Japan", "Canada", "Brazil"]
```

Print the total number of countries.

---

## Exercise 5: Add Elements

Create an array called `tasks`:

```js
["Learn HTML", "Learn CSS"]
```

1. Add `"Learn JavaScript"` to the end.
2. Add `"Set up VS Code"` to the beginning.

Print the updated array.

---

## Exercise 6: Remove Elements

Create an array:

```js
["pen", "pencil", "eraser", "sharpener"]
```

1. Remove the last item.
2. Remove the first item.

Print the updated array after each operation.

---

## Exercise 7: Check for a Value

Create an array:

```js
["cat", "dog", "rabbit", "bird"]
```

Check whether:

* `"dog"` exists
* `"fish"` exists

Print the results.

---

## Exercise 8: Find an Index

Create an array:

```js
["React", "Next.js", "Node.js", "PostgreSQL"]
```

Print the index of:

* `"Node.js"`
* `"Express"`

---

## Exercise 9: Copy an Array

Create an array:

```js
const original = ["apple", "banana"];
```

1. Create a copy using the spread operator.
2. Add `"orange"` to the copied array.
3. Print both arrays.

---

## Exercise 10: Combine Arrays

Create two arrays:

```js
const frontend = ["HTML", "CSS"];
const backend = ["Node.js", "PostgreSQL"];
```

Combine them into a new array called `skills`.

Print `skills`.

---

## Exercise 11: Use slice()

Create an array:

```js
["January", "February", "March", "April", "May"]
```

Create a new array containing only:

```js
["February", "March", "April"]
```

Print both arrays.

---

## Exercise 12: Use splice()

Create an array:

```js
["HTML", "CSS", "Java"]
```

1. Replace `"Java"` with `"JavaScript"`.
2. Insert `"Git"` between `"CSS"` and `"JavaScript"`.

Print the array after each operation.

---

## Exercise 13: Loop Through an Array

Create an array:

```js
["keyboard", "mouse", "monitor"]
```

Print each item using:

1. A `for` loop
2. A `for...of` loop

---

## Exercise 14: Arrays of Objects

Create an array named `students`:

```js
[
  { id: 1, name: "Sara" },
  { id: 2, name: "John" },
  { id: 3, name: "Ava" }
]
```

Print:

1. The name of the first student
2. The ID of the last student

---

## Challenge Exercise

Create an array:

```js
["apple", "banana", "orange"]
```

Perform the following operations in order:

1. Add `"grape"` to the end.
2. Remove `"banana"`.
3. Replace `"orange"` with `"mango"`.
4. Add `"kiwi"` to the beginning.

Print the final array.
